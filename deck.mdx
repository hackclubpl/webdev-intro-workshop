```typescript
addEfectToImages(image:object, effect:string): string;

Wejście:
    image - plik do zdjęcia lub URL, format png
    effect - reverse, contrast, darken, brighten
Wyjście:
    plik ze zdjęciem, przekonwertowany.

Jak uzyskać dostęp?
Przez http - GET https://server1/addEfectToImage?image=url&effect=effect
Przez pobranie paczki np. przez npm i import modułami ES6
```
